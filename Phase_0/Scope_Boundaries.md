## Why the boundary exists
Execution requires authority, contracts, legal review, and balance-sheet ownership. This repo is deliberately limited to what can be defined without those.

## Plain-English summary
This repo explains how to prepare for safe slowdown.  
It does not authorize real-world execution.  
When real cuts and contract changes begin, this repo is no longer the right tool.
